# Design as a Service

A landing page designed and developed during a 2-week internship. The design was created in Figma and implemented using Webflow.

- **Webflow:** [Preview](https://preview.webflow.com/preview/designasaservice?utm_medium=preview_link&utm_source=designer&utm_content=designasaservice&preview=c46d14f8d4248a2ed283183fda559a9a&workflow=sitePreview)
- **Figma:** [Design](https://www.figma.com/design/aQUYXE3ck7xjQMvvXbQJr2/Design-as-a-Service?node-id=1-6&p=f&t=Pdg7BYVoVLyQGumm-0)

## Project Overview

**Timeline:** 2-week internship  
**Role:** Designer & Developer  
**Tools:** Figma, Webflow

### What was done

- Designed the full UI/UX in Figma, including layout, typography, and color system
- Implemented the design in Webflow with responsive breakpoints
- Delivered a production-ready landing page within the internship timeframe
