A landing page designed and developed during a 2-week internship. Designed in Figma and implemented in Webflow using the Client First framework by Finsweet.

## Project Overview

**Timeline:** 2-week internship  
**Role:** Designer & Developer  
**Tools:** Figma, Webflow, Client First (Finsweet)

### What was done

- Designed the full UI/UX in Figma, including layout, typography, and color system
- Implemented the design in Webflow following the [Client First](https://www.finsweet.com/client-first) naming convention and structure by Finsweet
- Delivered a production-ready landing page within the internship timeframe
